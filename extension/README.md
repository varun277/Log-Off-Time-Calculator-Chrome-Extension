# Work Time Calculator Extension

## How to Install

1. **Download the files**: Download the `extension` folder from this Replit project to your computer.
2. **Open Chrome Extensions**:
   - Open Google Chrome.
   - Go to the URL: `chrome://extensions`
3. **Enable Developer Mode**:
   - Toggle the switch named "Developer mode" in the top right corner.
4. **Load Unpacked**:
   - Click the "Load unpacked" button in the top left.
   - Select the `extension` folder you downloaded.

## How to Use

1. Click the extension icon in your Chrome toolbar.
2. Enter your **Start Time** (defaults to current time).
3. Enter **Hours to Work** (e.g., 8 or 8.5).
4. Check "Notify me when done" if you want a browser notification.
5. Click **Calculate Logout Time**.
6. The extension will show you when you can leave!
