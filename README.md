# ⏳ Log Off Time Calculator – Chrome Extension

A simple and lightweight Google Chrome extension that helps users calculate their ideal log off time based on their check-in time and required working hours.

---

## 🚀 Overview

This extension allows users to:

- Enter the time they started working
- Enter the total number of hours required for the day
- Instantly calculate the exact time they can log off
- Enable notifications to get alerted when the workday is complete

It is designed to be simple, fast, and easy to use.

---

## ✨ Features

- Input office check-in time
- Enter required total working hours
- Automatic log off time calculation
- Optional notification alert when time is reached
- Clean and minimal interface
- Lightweight Chrome extension
